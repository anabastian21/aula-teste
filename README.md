\# 🧑‍💻 Primeira Atividade com Git



\## 📌 Objetivo

Esta atividade tem como objetivo aprender e aplicar as \*\*configurações iniciais do Git\*\*, entendendo os primeiros comandos necessários para utilizar controle de versão em projetos.



---



\## ⚙️ Configurações iniciais



Para começar a utilizar o Git, é necessário configurar o \*\*nome de usuário\*\* e o \*\*e-mail\*\*, que serão associados aos commits realizados.



\### Configurar nome de usuário

```bash

git config --global user.name "Seu Nome"
